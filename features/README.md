# Features

This directory holds the active source-of-truth docs for each major post-Slice-1 feature.

Use one doc per feature. Each feature doc owns:

- scope
- dependencies
- delivery gates
- public interface expectations
- QA proof and completion evidence

`master-plan.md` is the rollup. These feature docs are the detailed source of done.
